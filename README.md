# AI Content Factory (n8n)

## Overview
This project demonstrates a scalable AI content production system using n8n.

It automates the entire pipeline from idea generation to content-ready output.

## Features
- Topic generation via AI
- Script generation (LLM)
- Content structuring
- Media pipeline integration
- Automation via webhooks

## Workflow
Includes production-ready 
n8n workflow (workflow.json)

## Tech Stack
- n8n
- OpenAI API
- Webhooks
- JSON workflows⁷

## Workflow Structure
1. Input (topic / trigger)
2. AI generation
3. Structuring content
4. Output formatting
5. Delivery (file / webhook)

## Use Cases
- YouTube automation
- Content marketing
- AI media production
